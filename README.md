# Text to Number Converter

A simple Python project that converts numbers written in **English words** into their corresponding **numeric values**.

## Overview

This project demonstrates how English text numbers such as `"one"`, `"twenty five"`, or `"ninety"` can be converted into integers using a dictionary-based approach in Python. The code is lightweight, easy to understand, and suitable for beginners.

## Features

- Converts English text numbers to integers  
- Uses a simple dictionary mapping  
- No external libraries required  
- Easy to extend and modify  

## Example

```python
text_to_number("twenty five")
# Output: 25
